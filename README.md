# kafka-admin-ansible

## Задача
* Сделать бэкап текущей версии Debezium Connectors
* Распаковать новые версии в отдельные подкаталоги `/kafka/plugins/<connector-name>`. Kafka ищет плагины именно в подкаталогах `plugin.path,` а не в корне.
* Перезапустить сервис Kafka Connect, чтобы он подхватил новые JAR‑файлы. В миграционных заметках Debezium прямо указано, что обновление выполняется остановкой коннектора, заменой плагина и последующим запуском; после рестарта коннекторы продолжат с того же offset.

## Структура роли
```
roles/debezium_upgrade/
  defaults/main.yml
  files/
  tasks/main.yml
```
